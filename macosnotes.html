rsync settings

type:

rsync -vhaHE --progress --delete /Users/nikolask/Documents/ /Volumes/backup 2> ~/Desktop/rsyncErrors.txt

options:

  • v: verbose
  • h: human readable numbers
  • a: archive, to keep times, symlinks, permissions, groups, owners and traverse directories recursively
  • H: to keep any hard-link
  • E: to keep extended attributes
  • progress: to keep an eye on job progress
  • delete: removes any extraneous files on the destination
  • 2> ~/Desktop/rsyncErrors.txt: contain any errors during the copy and why they failed to copy

git settings

global user settings

    git config --global user.name "Nikos Kyriazis"
    git config --global user.email nikos.kyriazis@gmail.com

create .gitignore_global

    touch ~/.gitignore_global
    open .gitignore_global

add the following to ignore:

    # Logs and databases #
    ######################
    *.log
    *.sqlite

    # OS generated files #
    ######################
    .DS_Store
    .DS_Store?
    ._*
    .Spotlight-V100
    .Trashes
    Icon?
    ehthumbs.db
    Thumbs.db

add the following to include .gitignore_global

    git config --global core.excludesfile ~/.gitignore_global

and finally:

    git config --global color.ui true
    git config --global core.autocrlf input

at the terminal type:

    defaults write com.apple.screencapture disable-shadow -bool true
    defaults write com.apple.screencapture type  JPG
    killall SystemUIServer

Snappier Dock

open a terminal and type:

        defaults write com.apple.dock autohide-delay -int 0
        defaults write com.apple.dock autohide-time-modifier -float 0.4
        killall Dock

to restore default settings type:

        defaults delete com.apple.dock autohide-delay
        defaults delete com.apple.dock autohide-time-modifier
        killall Dock
