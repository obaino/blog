set up ssh / git

To create a pair of ssh keys: 
    ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
To add your key to the agent: 
    eval "$(ssh-agent -s)"
    ssh-add ~/.ssh/id_rsa

Copy the contents of ~/.ssh/id_rsa.pub to the new key field in GitHub.

To verify you have completed the setup properly 
    ssh -T git@github.com
To clone new repositories instead of using https:// 
    git clone git@github.com:USERNAME/Repo.git
To update existing repositories to use SSH instead of https 
    git remote set-url origin git@github.com:USERNAME/Repo.git

Setup Logitech Marble Mouse

Create /usr/share/X11/xorg.conf.d/50-marblemouse.conf and add the following lines: 
    Section "InputClass"
        Identifier  "Marble Mouse"
        MatchProduct "Logitech USB Trackball"
        MatchIsPointer "on"
        MatchDevicePath "/dev/input/event*"
        Driver "libinput"
        Option "ButtonMapping" "1 9 3 4 5 6 7 8 2"
        Option "ScrollButton" "8"
        Option "ScrollMethod" "button"
        Option "MiddleEmulation" "on"
